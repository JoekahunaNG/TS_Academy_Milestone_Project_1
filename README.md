# 🚀 Milestone Project 1: HarunaDev

A comprehensive showcase of fundamental HTML5 concepts and semantic web design principles, developed as part of the **TS Academy Software Development Programme**.

## 📖 Project Overview

This project serves as a cornerstone for demonstrating proficiency in "Plain Old Semantic HTML" (POSH). It encapsulates the core learning outcomes from the first six weeks of the TS Academy curriculum, focusing on accessibility, multimedia integration, and interactive data collection.

### ✨ Key Features

*   **Semantic Structure**: Organized using modern HTML5 tags (`<header>`, `<main>`, `<footer>`, `<section>`, `<article>`) for enhanced SEO and accessibility.
*   **Multimedia Integration**:
    *   **Smooth Video**: Autoplaying, muted, and looping video content.
    *   **Accessible Imagery**: Use of `<figure>` and `<figcaption>` for semantically associated visual content.
*   **Data Presentation**: A well-structured progress table utilizing `<thead>`, `<tbody>`, and `scope` attributes for unambiguously associated data headers.
*   **Interactive Forms**: A robust contact form with built-in HTML5 validation (lengths, required fields, and specific input types).
*   **Navigation**: Fragment identifiers for smooth internal page jumping (e.g., "Contact Us" linking directly to the footer form).

## 🛠️ Technology Stack

*   **Structure**: HTML5 (Semantic elements)
*   **Multimedia**: HTML5 Video and Figure/Image API
*   **Interactive**: HTML5 Forms & Validation
